CSS Table Reference

Text Align

Tag Name Description
left text-align: left Aligns the text to the left
right text-align: right Aligns the text to the right
centre text-align: centre Centers the text
justify text-align: justify Stretches the lines so that each line has equal width (like in newspapers and magazines)
initial text-align: initial Sets this property to its default value.
inherit text-align: inherit Inherits this property from its parent element.

Syntax Setting Description
